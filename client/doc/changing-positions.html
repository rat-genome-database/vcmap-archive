Changing Map Positions

Moving Maps

Maps can be moved around the VCMap window by clicking on the name of the map and dragging it to the location you wish that map to be. You will know you are dragging the map if there is a blue outline of the map moving around as you move your mouse. As you pass over other maps, the window will update with the current location of the map you are dragging. The map will be placed as soon as you release the mouse button.

Restoring Default Ordering

Restoring the default ordering of the maps will undo any changes you have done to the order of the maps. By using this option, the map layout will revert back to the default order that the maps were loaded. This can be done by selecting "Map -> Restore Positions" from the menu bar or by using the keyboard shortcut "CTRL + R".
Note: The switch will happen immediately and there is no message asking for confirmation.

Additional Information

  • A map must be visible in order to move it.
  • Hidden maps will change positions when they are affected by a visible map is move.
