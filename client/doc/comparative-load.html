What to do Next?

Now that you have successfully loaded multiple maps, you may:

  1. Load Annotation: Loading annotation data will display additionally features on all of the maps that you have loaded so far. You can load this data and explore the relationships between these features and where they exist on each map.
  2. Selecting Annotation: Features that have been identified as homologous features will display a connecting line between maps. When you select one of these features, the homologous features on other maps will be selected at the same time.
  3. Downloading Information: You can selectively download information from VCMap and open it in a spreadsheet application like Microsoft Excel. Follow these instructions to select a group of annotation and download filtered data in a personally customized format to your system's default spreadsheet application.
